◆ __HAL_RCC_ADC12_CONFIG

#define __HAL_RCC_ADC12_CONFIG ( __ADC12_CLKSOURCE__)
Valeur :
MODIFY_REG(RCC->CCIPR, RCC_CCIPR_ADC12SEL, (__ADC12_CLKSOURCE__))

Macro to configure the ADC12 interface clock.

Paramètres
__ADC12_CLKSOURCE__specifies the ADC12 digital interface clock source. This parameter can be one of the following values:
  • RCC_ADC12CLKSOURCE_NONE No clock selected as ADC12 clock
  • RCC_ADC12CLKSOURCE_PLL PLL Clock selected as ADC12 clock
  • RCC_ADC12CLKSOURCE_SYSCLK System Clock selected as ADC12 clock
Valeurs retournées
None

◆ __HAL_RCC_RNG_CONFIG

#define __HAL_RCC_RNG_CONFIG ( __RNG_CLKSOURCE__)
Valeur :
MODIFY_REG(RCC->CCIPR, RCC_CCIPR_CLK48SEL, (__RNG_CLKSOURCE__))

Macro to configure the SAI1 clock source.

Paramètres
__SAI1_CLKSOURCE__defines the SAI1 clock source. This clock is derived from the HSI, system PLL, System Clock or external clock. This parameter can be one of the following values:
  • RCC_SAI1CLKSOURCE_SYSCLK SAI1 clock = System Clock
  • RCC_SAI1CLKSOURCE_PLL SAI1 clock = PLL "Q" clock
  • RCC_SAI1CLKSOURCE_EXT SAI1 clock = EXT
  • RCC_SAI1CLKSOURCE_HSI SAI1 clock = HSI
Valeurs retournées
None

Macro to configure the RNG clock.

Note
USB and RNG peripherals share the same 48MHz clock source.
Paramètres
__RNG_CLKSOURCE__specifies the RNG clock source. This parameter can be one of the following values:
  • RCC_RNGCLKSOURCE_HSI48 HSI48 selected as RNG clock for devices with HSI48
  • RCC_RNGCLKSOURCE_PLL PLL Clock selected as RNG clock
Valeurs retournées
None
